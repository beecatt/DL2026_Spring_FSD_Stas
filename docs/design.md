# Design

## Общая идея

Приложение предназначено для генерации QR-кодов с возможностью кастомизации и сохранения.

## Основные экраны

1. Форма создания QR:
   - ввод текста
   - выбор цветов
   - загрузка изображения

2. Preview:
   - отображение QR-кода в реальном времени
   - кнопка скачивания

3. История:
   - список сохранённых QR
   - повторное использование
   - удаление

## UX решения

- простая и понятная форма
- мгновенный preview
- единый стиль кнопок
- адаптивность

## Технические решения

- разделение frontend/backend
- хранение файлов отдельно от БД
- использование sessionStorage для черновика
